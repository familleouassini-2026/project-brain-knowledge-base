# ADR-001 — Markdown comme source de vérité

## Décision
La base de connaissance officielle est écrite en Markdown et versionnée dans Git.

## Raisons
- Lisible par humains et IA.
- Facile à versionner.
- Permet de générer PDF, site documentaire et contexte IA.

## Conséquence
Le PDF est une publication. Il n’est jamais modifié directement.
