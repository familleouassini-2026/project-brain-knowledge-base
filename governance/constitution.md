# Constitution Project Brain

## Principes fondateurs

1. La documentation est la source de vérité.
2. Le code est une implémentation, pas le patrimoine principal.
3. L’IA propose, l’humain décide.
4. Aucun changement majeur sans analyse d’impact et ADR.
5. Le plus petit changement compatible est privilégié.
6. Chaque fonctionnalité doit être explicable, traçable et testable.
7. Le projet doit pouvoir être repris par une autre équipe ou une autre IA.
