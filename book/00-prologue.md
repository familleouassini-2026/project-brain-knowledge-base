# Prologue — Avant le premier diagramme

Le projet n’a pas commencé par une architecture logicielle. Il a commencé par une idée simple : créer un site web.

Puis une question est apparue : comment éviter de perdre la maîtrise lorsque le projet grandit, change, se documente, se développe et implique plusieurs parties prenantes ?

C’est à ce moment-là que l’idée de Project Brain est née : non pas comme un outil qui décide, mais comme un gardien de la cohérence.
